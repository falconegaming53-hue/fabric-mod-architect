# MC Fabric Mod Architect 🔫
Automatic Fabric Mod Maker for Minecraft 1.21.4

## Features
- 🎯 Browser Mode (no API key needed — uses ChatGPT/Claude/Gemini free)
- 🩹 Self-Healing (auto-fixes compilation errors)
- 🧵 Fabric Loom + Gradle auto-setup (no system Gradle needed)
- 📦 Auto-generates: build.gradle, fabric.mod.json, mixin JSON, lang files
- 🔧 Auto-fixes: Identifier.of(), Registries class, Data Components

## Quick Start
1. Install Python 3.8+ + Java 21
2. Run: `python fabric_mod_architect.py`
3. Select Browser Mode → describe your mod → paste AI response → done!

## Requirements
- Python 3.8+
- Java JDK 21
- `pip install requests`
- Minecraft Java Edition (to test mods)
